# Ланцюжковий калькулятор

Консольний калькулятор на Ruby з реалізацією патерну **Command**. Зберігає поточне значення і виконує над ним операції
одну за одною (ланцюжком).

## Вимоги

- Ruby
- Bundler

## Встановлення

```bash
bundle install
```

## Запуск

```bash
ruby main.rb
```

Спершу вводиться початкове число, далі — операції по одній. Для виходу — `exit`.

## Команди

Операції, що **запитують ще одне число**:

| Команда  | Дія                          |
|----------|------------------------------|
| `+`      | додавання                    |
| `-`      | віднімання                   |
| `*`      | множення                     |
| `/`      | ділення                      |
| `mod`    | остача                       |
| `pow`    | піднесення до степеня        |
| `primes` | прості числа до заданого max |

Операції **без додаткового числа**:

| Команда | Дія                  |
|---------|----------------------|
| `++`    | інкремент            |
| `--`    | декремент            |
| `sqrt`  | квадратний корінь    |
| `sin`   | синус                |
| `cos`   | косинус              |
| `tan`   | тангенс              |
| `ctan`  | котангенс            |
| `exp`   | експонента           |
| `ln`    | натуральний логарифм |
| `!`     | факторіал            |
| `mw`    | записати в пам'ять   |
| `mr`    | зчитати з пам'яті    |
| `push`  | покласти в стек      |
| `pop`   | дістати зі стеку     |
| `exit`  | вихід                |

Тригонометрія працює в градусах.
